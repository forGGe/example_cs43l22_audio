# theCore _CS43L22 audio_ example

## Quick start

1. Install theCore:

   ```
   pip install tcore
   tcore bootstrap
   ```

1. Download this example:

   ```
   tcore init https://github.com/theCore-embedded/example_cs43l22_audio
   ```

1. Compile:
   
   ```
   cd example_cs43l22_audio
   tcore compile --target stm32f4_disc
   ```
